**reset_usb** is a Linux command-line tool to reset a usb device
based on Vendor ID and Product ID

## Installation

Prerequisites:

libusb1.0

then download

```
cd ~
sudo git clone https://github.com/Samaelson/reset_usb
cd reset_usb
sudo make
```

## Usage

```
sudo ./reset_usb 0x0403 0x6001
```
